## h5 Fuzzy

### x) Lue/katso/kuuntele ja tiivistä. 

[Karvinen 2023: Find Hidden Web Directories - Fuzz URLs with ffuf](https://terokarvinen.com/2023/fuzz-urls-find-hidden-directories/)

- Ffuf on työkalu, jolla voi käydä läpi esimerkiksi nettisivuston alasivuja, ja löytää sieltä jotain kiinnostavaa
- Itse Ffufin lisäksi tarvitaan sanalista, joita on valmiita tai sellaisen voi tehdä itsekin
- Komento ``ffuf -w common.txt -u http://127.0.0.2:8000/FUZZ`` kokeilee common.txt-listan sanoja sivuston URL:n perään 
- Hakutuloksia voidaan filtteröidä erilaisilla komennoilla, kuten sivun koko "fs" tai sanojen lukumäärä "fw"

[Hoikkala 2023: ffuf README.md](https://github.com/ffuf/ffuf/blob/master/README.md)
- Ffufin kanssa voi käyttää ulkoista mutatoria, joka generoi fuzzattavia arvoja
- Esimerkiksi ``ffuf --input-cmd 'radamsa --seed $FFUF_NUM example1.txt example2.txt'`` käyttää radamsa-mutatoria, joka generoi sille annettujen tekstitiedostojen avulla uusia arvoja, joita voidaan kokeilla kohteen fuzzaamisessa
- Komentorivillä annetut vaihtoehdot ylikirjoittavat konfiguraatiotiedostoon laitetut komennot, paitsi jos samaa vaihtoehtoa voi olla useampi, koska silloin ne molemmat otetaan huomion
- Enter-näppäimellä voi pysättää prosessin ja avata interaktiivisen tilan, jossa voi esimerkiksi muokata filttereitä 
- Filttereiden muokkaaminen vaikuttaa jo ajettuihin tuloksiin
- Restart-komennolla voi aloittaa muokatun ajon alusta

### a) Fuzzzz. Ratkaise dirfuz-1 artikkelista Karvinen 2023: [Find Hidden Web Directories - Fuzz URLs with ffuf](https://terokarvinen.com/2023/fuzz-urls-find-hidden-directories/).

Asensin dirfuzt-1:n ja muutin sen oikeudet ajettavaksi.

![](/images/h5_1dirasennus.png)

Ajoin sen ohjeissa annetulla komennolla.  
![](/images/h5_2ajoindirin.png)

Nyt localhost:8000:ssa näkyi tehtävä.  
![](/images/h5_3dirselaimessa.png)

Asensin ffuf-työkalun komennolla ``sudo apt-get install ffuf`` ja sitten latasin sanalistan komennolla ``wget https://terokarvinen.com/2023/fuzz-urls-find-hidden-directories/Miessler-etal-2023-SecLists/common.txt``.

Ffufin ohjesivut näkyvät komennolla ``ffuf`` tai ``ffuf -help``.
 
![](/images/h5_4ffufhelp.png)

Laitoin Kalin nettiyhteyden pois päältä ja testasin, että pingit eivät toimi ja sitten fuzzasin kohteen komennolla ``ffuf -w common.txt -u http://127.0.0.2:8000/FUZZ``. Lähes kaikissa tuloksissa näkyi olevan http response 200 eli OK, ja ne olivat samankokoisia, joten tuskin niissä on mitään mielenkiintoisia.

![](/images/h5_5ffufsekava.png)

Filteröin yleisen sivukoon pois optiolla -fs, ja nyt tuloksissa näkyi vain muutama sivu.

![](/images/h5_6filtersize.png)

Tehtävänannossa mainittu admin-sivu löytyi osoitteesta /wp-admin ja versionhallintaan liittyvä sivu osoitteesta /.git

![](/images/h5_7flag1.png)

![](/images/h5_8flag2.png)

Huomautus: Olin tehnyt saman tehtävän eräällä aiemmalla kurssilla, mutta tein sen tässä uudelleen, enkä katsonut vanhoista muistiinpanoistani mitään apua.

### b) Fuff me. [Asenna FuffMe-harjoitusmaali. Karvinen 2023: Fuffme - Install Web Fuzzing Target on Debian](https://terokarvinen.com/2023/fuffme-web-fuzzing-target-debian/)

Asensin docker.io:n ``sudo apt-get install`` ja kloonasin ohjeissa annetun repon ``git clone https://github.com/adamtlangley/ffufme``.

Rakensin imagen komennolla ``sudo docker build -t ffufme .``

![](/images/h5_9dockerbuild.png)

Sen jälkeen ajoin imagen ``sudo docker run -d -p 80:80 ffufme``. Optio ``-d`` ajaa kontin taustalle, ettei se ota terminaalia käyttöönsä. ``-p 80:80`` reitittää kontin sisällä olevan portin 80 isäntäkoneen porttiin 80 (isäntä:kontti). Lähde: [https://ouro.fi/docker-perusteet-komentorivi/](https://ouro.fi/docker-perusteet-komentorivi/).

![](/images/h5_10dockerrun.png)

Tehtävät näkyivät nyt localhostissa. Latasin tehtävässä mainitut sanalistat kansioon ~/wordlists/.

### f) No 404 Status

Tehtävässä käytettiin sivukoon filtteriä "-fs 669", koska suuri osa tuloksista oli sen kokoisia "sivua ei löydy" -vastauksia, mutta niistä puuttui 404 header, jolloin ffuf ei pystynyt poistamaan niitä tuloksista automaattisesti.

### g) Param Mining

Tehtävän osoitteessa localhostissa näkyi teksti "Required Parameter Missing". Kohde fuzzattiin niin, että osoitteen perässä oli ``?FUZZ=1``.

![](/images/h5_21tulos.png)

Tuloksesta löytyi parametri "debug", jolla löytyi oikea sivu.

![](/images/h5_22para.png)

### h) Rate Limited
Tehtävän ohjeessa neuvottiin käyttämään komentoja, jotka hidastavat fuzzaamista, koska kohde rajoittaa pyyntöjen määrää. ``-t 5`` luo työkalusta viisi versiota ja ``-p 0.1`` odottaa jokaisen pyynnön kohdalla 0,1 sekuntia. Lisäksi ``-mc`` näyttää vain http-statukset 200 ja 429.

![](/images/h5_23ohje.png)

Olin kopioinut komennon suoraan tehtävästä, ja se antoi pelkkiä virheitä, eikä yhtään tulosta.

![](/images/h5_24eitoimi.png)

Googlasin ongelman ja löysin [keskustelusivun kommentin tehtävästä](https://github.com/BuildHackSecure/ffufme/issues/1). Siinä puhuttiin kohteen osoitteesta. Katsoin edellisestä tehtävästäni, että niissä oli käytetty osoitteessa localhostia, ja sitä käyttämällä tehtävä onnistui.

![](/images/h5_25onnistui.png)


![](/images/h5_26oracle.png)
